Breast Cancer Detection using Deep Learning

Learn More

The reason why

The reason why behind this decision is quite obvious. Cancer is the disease that is responsible for a huge percentage of annual deaths and still we have not found a way to significantly face it. So after a bit of research, breast cancer was the choice that I made in order to create a prediction model for.


To complete my solution on the problem, a Convolutional Neural network (CNN) was built and trained properly in an existing database. The particular agent is used to predict cancer from histopathology images, which in my case come from the dataset, and to output the results.

Model Architecture

A sequential neural network was created using Keras from Tensorflow.

The model consists of successive Convolutional - MaxPooling layers, each of which directs the information to the next levels, followed by a Flatten and a Dense layer. In the last Dense layer, the final decision is calculated.

The model was structured using the Adam optimizer and the binary cross-entropy error function.
